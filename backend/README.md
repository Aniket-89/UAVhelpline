### Python and pip Installation

1. Install python

-Linux/Mac: 
```bash 
sudo apt-get install python3
```

-Windows: See here: https://www.python.org/downloads/release/python-3131/

2. Pip installation

-Linux/Mac:
```bash
python -m ensurepip --upgrade
```

-Windows:
```bash
py -m ensurepip --upgrade
```
